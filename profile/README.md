# ✨ Key Features

- 🪐 **Empire Profile Manager:** Save and switch between different galaxy builds  
- 🧩 **Mod Organizer:** Manage total conversions, AI, and gameplay mods  
- 💾 **Auto Save Backup:** Protect galaxy saves and progression  
- 🛡 **Safe Configuration Layer:** No modification of core game files  
- 🌍 **Cross-Platform Support:** Windows, macOS, Linux compatible  
- ⚡ **Performance Optimization:** Improve stability in large late-game galaxies  
- 🌌 **Galaxy Scenario Switcher:** Quickly load different empire setups  
- 🔄 **Auto Updates:** Keeps mod environment up to date  
- 🎁 **Open-Source & MIT Licensed:** Free and community-driven
